Sales Report Dashboard
Project Overview

This project presents an interactive Sales Report Dashboard developed using Power BI to analyze sales performance across different production lines and products. The dashboard converts raw sales data into meaningful insights that help track trends, evaluate product performance, and support data-driven business decisions.

Dashboard Preview

Key Metrics

Average Monthly Sales: 2.60M

Total Sales by Production Line: 15.43M

Units Sold: 5.5M

Dashboard Features

KPI Cards: Displays key business metrics such as average monthly sales, total sales, and units sold.

Sales by Production Line: Compares total sales and units sold across different production lines.

Month-over-Month Sales Analysis: Tracks monthly sales performance and trends.

Sales Contribution: Shows percentage contribution of each production line to overall sales.

Product-Level Analysis: Detailed table showing sales performance for individual products.

Year-over-Year Trend: Visualizes sales growth trends across different years.

Interactive Filters: Allows users to filter data by Year, Production Line, and Item.

Tools & Technologies

Power BI – Data visualization and dashboard development

DAX – Calculated measures and KPIs

Excel / SQL – Data preparation and transformation

Key Insights

Certain production lines contribute significantly more to total sales.

Sales trends vary across months, indicating seasonal demand patterns.

Product-level analysis helps identify top-performing and low-performing items.

Year-over-year analysis highlights overall business growth.
